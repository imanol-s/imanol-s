## ‚ú® Solution

- **Data Processing:** Cleaned and analyzed 20,000+ crime and housing records in R, handling missing values and data inconsistencies
- **Feature Engineering:** Created features from crime density, location factors, and property characteristics
- **Model Development:** Built and compared multiple models including linear regression, random forest, and k-NN classifiers
- **Geospatial Analysis:** Developed ggplot2 dashboards to visualize geographic and statistical correlations across Dallas neighborhoods
- **Spatial Insights:** Identified patterns showing how crime rates correlate with housing costs in different areas
- **Documentation:** Delivered comprehensive technical report documenting data preprocessing, model evaluation, and interactive visualizations
